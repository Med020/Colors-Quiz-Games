# Colors-Quiz-Games

This project is a dynamic web application for quiz games, built using Django as the backend framework.

## Features

- Interactive quiz games
- Real-time score tracking
- Various quiz categories
- Leaderboards

## Technologies Used

- Django
- HTML/CSS
- JavaScript
